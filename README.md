# Worgy Freestyle

Генератор слов для тренировки фристайла с битами.

## Требования

- [Node.js](https://nodejs.org/) версии 18 или выше

## Установка и запуск

1. **Установи зависимости:**
   ```bash
   npm install
   ```

2. **Запусти приложение в режиме разработки:**
   ```bash
   npm start
   ```

## Сборка .exe файла

**Portable версия (один exe файл, не требует установки):**
```bash
npm run build
```

**Инсталлятор (с возможностью выбора папки установки):**
```bash
npm run build:installer
```

Готовые файлы появятся в папке `dist/`.

## Иконка приложения

Чтобы добавить свою иконку, положи файл `icon.ico` (256x256 пикселей) в корень проекта.

Можно создать иконку онлайн: https://convertico.com/

## Горячие клавиши

- **Пробел** — следующее слово (когда бит не играет)

## Возможные проблемы

### PowerShell не даёт выполнить npm

Если появляется ошибка "выполнение сценариев отключено", запусти PowerShell от имени администратора и выполни:
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

Или используй обычную командную строку (cmd) вместо PowerShell.

### Нет звука

Убедись, что папка `audio/` с файлами битов находится рядом с index.html.
